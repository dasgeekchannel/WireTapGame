WireTap is a text based adventure (work in progress) python based game. 
The story is a cyberpunk science fiction piece.



# Install dependencies
sudo dnf install pip
pip3 install climage colorama

# Clone repo and execute
git clone https://github.com/dasgeekchannel/WireTapGame
cd WireTapGame/
python wiretapv2.py
